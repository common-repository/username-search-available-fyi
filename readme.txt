Search for available usernames on Instagram, Twitter, TikTok, Facebook, Reddit, and YouTube.

== Description ==

Username Search is a Gutenberg block plugin that allows visitors to check the availability of a username on a social network.


### Supported Networks:

* Instagram
* Twitter
* Facebook
* TikTok
* Snapchat
* Reddit
* YouTube


### Privacy

The blocks included with this plugin use iFrames to embed a form into a page. The form is hosted by [Available.FYI](https://available.fyi).

[Available.FYI](https://available.fyi) is a Software as a Service product.

Available.FYI does not track visitors or user of this plugin. Information entered into the search form, along with IP addresses and web browser information, will be transmitted to the Available.FYI servers but the data will not be stored or logged. Available.FYI does not sell or rent user data.

You can read the [Available.FYI privacy policy here](https://available.fyi/privacy).
